# Startup India 2026

Empowering India’s Future with AI-Powered Skill Development & Internships.

## Features
- AI-Driven Career Hub
- Role-based Admin & Employee Dashboards
- Government & Industry Recognized Certifications
- Secure Payment Integration (Razorpay)
- Robust Authentication (Clerk)

## Tech Stack
- **Frontend**: Next.js 16 (App Router)
- **Backend**: Next.js Serverless Functions
- **Database**: MongoDB Atlas
- **Auth**: Clerk
- **AI**: Google Gemini AI
- **Styling**: Tailwind CSS

## Getting Started

First, install dependencies:
```bash
npm install
```

Then, run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Deployment
Deploy easily on Vercel by connecting your GitHub repository and configuring the necessary environment variables.
